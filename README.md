<h3>epiFi Android Assignment</h3>

This app is built in <b>Kotlin</b> with <b>MVVM</b> architecture design pattern along with Coroutines and Dependency Injection. 

<b>Libraries used:</b>
1. ViewModel
2. LiveData
3. DaggerHilt
4. Retrofit
5. GSON
6. Glide
7. Coroutines
8. Data Binding
9. Navigation Framework
10. Pagination
